# GitをCLIからを始まる

**Gitは触って見ないと分かり辛い** と思います。

本人は今までgitを見るだけか、SourceTreeだけ使うのがほとんどでした。
もっと詳しくGitを学習し、コマンドでも慣れたいと思います。

## 1. Gitって何ですか?

* 無料でオープンソース 分散形 **バージョン管理システム** 
* 開発テーマは 「 **速さ**」 
* gitは英語のスラングで「バカ」、「マヌケ」の意味がある
